# 怪物猎人：荒野 - 差分管理器 (Armor Variant Manager)

这是一款基于 REFramework 开发的《怪物猎人：荒野》(Monster Hunter Wilds) Lua 脚本 Mod。它的核心目的是让玩家能够精细化管理防具的外观，并将这些外观配置保存为预设。

## 前言
有时候做外观 Mod 可能会想要多几种配色，多几种穿搭方式，于是一个 Mod 可能就会分成很多个版本，这样 Mod 作者和玩家都十分不方便，于是便参考了 MDF-XL 的代码做了这个工具。原理与 MDF-XL 一致，通过显示/隐藏当前材质来控制对应的模型的显示。MDF-XL 功能强大但过于繁琐，只是使用这个功能的话对玩家十分不友好，并且不支持多人状态。

## ✨ 核心功能
* 可以将当前装备选中的材质保存为预设，可以将预设设为默认。
* 装备以当前身体（Body）部位为准，保存的预设会在 `reframework\data\ArmorVariantManager` 下，以当前身体部位 ID 命名，比如 `ch03_060_0002.json`。做好适配的话，玩家上手就可以在游戏内直接切换差分。
* 多人模式下，别的玩家会加载默认的预设（如果有）。

## 💡 使用说明
1. 启动游戏并打开 REFramework 面板。
2. 找到 `Armor Variant Manager` 脚本界面。
3. **管理外观**：在下方勾选或取消勾选对应的部件/材质，点击 `保存预设`。
4. **设置变身**：通过全新的 `变身管理` 模块，你可以让角色在特定条件下自动切换外观预设。
   * **工作模式**：支持“单选模式”（仅一种条件生效）和“条件并行模式”（多种条件同时生效时，系统将按预设的优先级进行智能调度）。
   * **触发条件类型**：
     * **生命值 (HP)**：根据角色的血量百分比阈值触发对应的预设（爆衣啦）。
     * **受击触发 (Damage)**：角色受到攻击时触发。可配置“延迟触发”、“循环变身”以及“链式变身”（即多个预设随时间推移自动按顺序切换）。
     * **武器状态 (Weapon)**：根据武器的“拔刀/收刀”状态自动切换预设，并支持按具体的武器类型进行过滤，使变身仅对特定武器生效。支持的武器如下：
       * **大剑 (Great Sword)**：支持按大剑**蓄力等级**、以及**蓄力招式类型**（上捞斩、空斩等）。
       * **太刀 (Long Sword)**：支持根据太刀的**气刃等级**（无、白、黄、红）。
       * **双刀 (Dual Blades)**：支持根据双刀的**鬼人化状态**。
       * **大锤 (Hammer)**：支持按大锤的**蓄力等级**。
       * **斩击斧 (Switch Axe)**：支持按当前的**剑/斧形态**。
       * **盾斧 (Charge Blade)**：支持按当前的**剑/斧形态**。
       * **操虫棍 (Insect Glaive)**：支持根据当前吸取的**精华颜色/强化状态**（红、白、橙）。
       * **弓箭 (Bow)**：支持根据弓箭的**蓄力等级**。
   * **操作步骤**：
     1. 展开“变身管理”面板，选择你想要启用的触发条件。
     2. 点击对应条件下的“添加规则”或“添加节点”。
     3. 设定该规则的触发阈值（如血量 < 50% 或 状态 = 拔刀/收刀，太刀气刃等级 = 无/1/2/3等等）。
     4. 将你保存好的“预设名称”绑定到该节点上，即可在游戏内自动生效。

## 📌 更新日志
* **V3.0.0**：引入全新的“变身管理”系统，支持基于条件（生命值、受击、武器拔刀/收刀，太刀气刃等级，大剑蓄力状态与等等武器）的动态外观切换。
* **V2.1.1**：修复了多人模式下的一些bug
* **V2.0.0**：
  1. 新增分组功能，可以将选中的选项抽离出一个分组，可以为新的分组单独配置预设。
  2. 去掉了加载按钮，现在选中即加载。
  3. 开放性能配置选项，可按照说明结合实际cpu性能调整，该选项直接关联着切换装备后加载预设的速度。
* **V1.2.1**：修复潜在的bug，优化性能
* **V1.2.0**：增加自动查找预设按钮，可以从json列表里查找匹配的预设，用于盒子等工具修改套装后恢复预设功能
* **V1.1.0**：修复主菜单/存档界面不生效的问题
* **V1.0.0**：发布
