# 怪物猎人：荒野 - 差分管理器 (Armor Variant Manager)

这是一款基于 REFramework 开发的《怪物猎人：荒野》(Monster Hunter Wilds) Lua 脚本 Mod。它的核心目的是让玩家能够精细化管理防具的外观（Mesh 与材质），并将这些外观配置保存为预设。

## 前言
有时候做外观 Mod 可能会想要多几种配色，多几种穿搭方式，于是一个 Mod 可能就会分成很多个版本，这样 Mod 作者和玩家都十分不方便，于是便参考了 MDF-XL 的代码做了这个工具。原理与 MDF-XL 一致，通过显示/隐藏当前材质来控制对应的模型的显示。MDF-XL 功能强大但过于繁琐，只是使用这个功能的话对玩家十分不友好，并且不支持多人状态。

## ✨ 核心功能
* 可以将当前装备选中的材质保存为预设，可以将预设设为默认。
* 装备以当前身体（Body）部位为准，保存的预设会在 `reframework\data\ArmorVariantManager` 下，以当前身体部位 ID 命名，比如 `ch03_060_0002.json`。做好适配的话，玩家上手就可以在游戏内直接切换差分。
* 多人模式下，别的玩家会加载默认的预设（如果有）。

## 💡 使用说明
1. 启动游戏并打开 REFramework 面板。
2. 找到 `Armor Variant Manager` 脚本界面。
3. **管理外观**：在下方勾选或取消勾选对应的部件/材质，点击 `保存预设`。
4. **设置变身**：在 `变身管理` 模块中，选择触发条件（生命值或武器状态，或条件并行），添加条件节点，并将刚刚保存的预设绑定到节点上即可。
